# ekolivs-oms
Ekolivs order management system

## Setup

Run make command:

```bash
make setup
```

- https://grpc.io/docs/protoc-installation/
- https://grpc.io/docs/languages/go/quickstart/

## Develop



Start the frontend. In `frontend` run

```bash
pnpm # installs dependencies
pnpm dev # starts dev server
```
